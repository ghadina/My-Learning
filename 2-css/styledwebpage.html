Absolute Positioning

Ignores the flow of the page entirely but moves with the page, it is relative to the top left of the page

fixed Positioning

Ignores the flow of the page entirely and does not move with the page, keeps the element fixed even when scrolling

Sticky Positioning

Ignores the flow of the page entirely and does not move with the page just like fixed, works like relative until scrolled past a certain threshold which is then treated as fixed

Static Positioning

This is the default value an element would have if you don't declare a position value for it and may be used to forcefully remove other positioning.

Margin
